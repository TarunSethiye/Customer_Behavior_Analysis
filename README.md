## 📌 Overview
This project analyzes customer shopping behavior to uncover patterns in purchasing, customer segmentation, and product performance.  

The goal is to help businesses make **data-driven decisions** by identifying:
- Spending trends  
- Customer segments  
- Product preferences  
- Subscription behavior  

---

## 📂 Dataset
- **Rows:** 3,900  
- **Columns:** 18  

### Key Features:
- 👥 Customer: Age, Gender, Location, Subscription  
- 🛒 Purchases: Product, Category, Amount, Season  
- 📊 Behavior: Discounts, Ratings, Frequency, Shipping  

### Data Cleaning:
- Handled missing values (Review Ratings)  
- Standardized column names  
- Removed redundant columns
